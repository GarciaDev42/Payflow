<h2 align="center">Sobre 📖</h2>
   
<p>
   O projeto PayFlow foi desenvolvido na Next Level Week Together 💜💚. Esse evento incrível traz um app cheio de funcionalidades diferentes, visando aprender na prática o máximo possível, além da intensa rotina de estudos, muitas conexões e networking. <br>
   O PayFlow se trata de um app mobile feito em Flutter 💙 para Gerenciamento de Boletos, contendo recursos como o uso de câmera e galeria, Machine Leaning com MLKit, Firebase Core e SignIn, Animações e Estilizações Personalizadas, entre vários outros pontos, como o uso do SharedPreferences. <br>
</p>

<h2 align="center">Como Usar 🤔</h2>

   ```
   - Clone esse repositório:
   $ git clone https://github.com/GarciaDev42/Payflow

   - Entre no diretório:
   $ cd Payflow

   - Instale as dependências:
   $ flutter pub get

   - Inicie o app: 
   $ flutter run
   ```
